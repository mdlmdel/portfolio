Featured Projects

SentiCord: Full-stack web application
SentiCord is a full-stack web application that uses the Watson AlchemyLanguage API to perform sentiment analysis on entities as they appear on the home pages of the Harvard Business Review, New York Times, Economist, Wall Street Journal, and more. Users can perform searches and save time-stamped versions of their reports. The reports record records of the searched terms, calculate the average sentiment score, and record the date the report was run and saved.


Technologies used: Node, Javascript, JQuery, AJAX, HTML, CSS, IBM Watson AlchemyLanguage API.
